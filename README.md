## 📋 Обо мне

Системный аналитик с опытом сбора и формализации требований к ПО, проектирования баз данных, документирования бизнес-процессов и взаимодействия с командами разработки.
 
Владею инструментами системного анализа:
 
-  **Нотации и моделирование:** UML, BPMN, IDEF0, C4, ER-диаграммы, Use Case, User Story
-  **API:** REST API, OpenAPI/Swagger, Postman, JSON
-  **Базы данных:** SQL, PostgreSQL, DBeaver
-  **Управление проектами:** Jira, Confluence, Agile
-  **Визуализация и дизайн:** Draw.io, Miro, Figma, PlantUML, Camunda
-  **Прочее:** Python, Excel (VBA, Power Query), GitHub
