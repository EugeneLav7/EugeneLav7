# Портфолио - Системный аналитик

Репозиторий содержит мои учебные и pet-проекты, выполненные в рамках профессиональной переподготовки и самостоятельной практики в области системного анализа.

Каждый проект - это набор аналитических артефактов: от сбора требований и моделирования предметной области до проектирования API и описания структуры данных.

## Проекты

| Проект | Описание | Ключевые артефакты |
|--------|----------|--------------------|
| [Trip Plan](./trip-plan/) | Мобильное приложение-спутник путешественника для туристического агентства | SRS, OpenAPI 3.0, ER-диаграмма, Use Cases, Спецификация (SRS) |

> Список проектов будет пополняться.

## Стек и инструменты

- **Нотации и стандарты:** UML (Use Case), ERD, IEEE 830 (SRS)
- **API-проектирование:** OpenAPI 3.0.3 (Swagger)
- **Базы данных:** PostgreSQL, SQL (DDL/DML)
- **Инструменты:** Draw.io, Swagger Editor, Markdown, Git
